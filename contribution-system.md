# 📊 공헌도 및 주식 시스템

## 🔎 핵심 개념
서버 내 유저 경쟁을 '공헌도' 수치로 시각화하여, 유저의 채집/기여/기술 투자 행동을 랭킹으로 연결합니다.

## 💡 공헌도 증가 방식
- 채집 시 해당 자원의 시세에 비례한 공헌도 증가  
- 특별 이벤트 참여 시 보너스 공헌도  
- 공헌도 '감소 아이템'을 이용한 방해 요소

## 📈 주식 시스템
- 작물/광물 주식은 AI 뉴스에 의해 실시간 시세 변동  
- 유저는 '투자자'로서 시세 예측 기반 전략 가능

## 🛠️ 기술
- 데이터 추적용 로그 DB  
- 주식 UI 연동 (명령어 + 책 GUI)
